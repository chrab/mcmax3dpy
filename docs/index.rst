mcmax3dpy
*********

The python package provides routines for reading and plotting the output MCMax3D models.
This is still a very rudimentary package. But everyone is welcome to contibute.


Installation
============

The package can be installed from a github repository. 
For details see https://github.com/chrab/mcmax3dpy

.. Their you will also find a binder like which allows you to try some features of 
   the package without installing anything.

..	Using Jupyter
	=============
	
	Most mcmax3dpy routines can also be used within a Jupyter notebook.
	In this :download:`example notebook <./jupyter_example.ipynb>` it is shown how to use 
	the reading a plotting routines for a single model. 
	
	This feature is not well tested yet. It worked with python 3.6 as part of the anaconda distribution using 
	the local Jupyter server.
